## 案件概要

- **案件番号**: ANK0000639199
- **業種**: ヘルスケア・医療IT
- **プロジェクト名**: 医療業界向けクラウドインフラ構築

## 作業内容

医療業界向けクラウドインフラ構築案件。

### 主な作業

- AWSを中心としたクラウドインフラの設計、構築、運用、保守
- AIモデル運用のためのMLOpsパイプライン構築、改善
- IaC(Terraform、CloudFormationなど)による環境構築の自動化
- コンテナ(Docker、ECS、EKS)ベースの運用管理
- 監視、ログ設計、アラート基盤などのSRE活動
- セキュリティ要件、医療データに関するガバナンスへの対応
- 高負荷処理が必要なAI向けHPC(High Performance Computing)環境の運用(該当する場合)
- 新規プロダクトのインフラ設計への技術選定、提案

## 環境

- AWS
- Terraform
- Docker

## 要求スキル

### 【必要】

- AWSでのクラウドインフラ構築や運用経験(5年以上)
- MLOps構築や運用経験
- IaCの知見や運用経験
- コンテナ技術の知見や運用経験
- SREに関する知見

### 【尚可】

- HPCの構築や運用経験
- 医療データを扱うシステムの運用経験
- セキュリティやガバナンスに関する知見
- 新規プロダクトのインフラアーキテクチャ設計経験
