# 商談用資料

GenSparkで作成したPowerPoint・画像ファイルを格納するディレクトリです。

## ファイル配置ガイド

| ファイル名 | 内容 |
|-----------|------|
| `医療AIプラットフォーム提案資料.pptx` | メインの提案資料（10枚） |
| `システム構成図.png` | AWS構成図 |
| `セキュリティ構成図.png` | セキュリティ対策の図 |
| `コスト見積もり.xlsx` | コスト詳細（Excel） |
| `エグゼクティブサマリー.pptx` | 経営層向け1枚資料 |

## 作成手順

1. [GenSpark構成図プロンプト.md](../GenSpark構成図プロンプト.md) からプロンプトをコピー
2. GenSparkに貼り付けて資料を生成
3. ダウンロードしたファイルをこのディレクトリに配置
4. `git add .` → `git commit` → `git push`
